# 과제 1 풀이 및 증명

#### 1. (연습문제 1.1-3) 합성함수의 정의역
* **$f(g(x))$:** $f(1/x) = \sqrt{\frac{1}{x}+1} = \sqrt{\frac{1+x}{x}}$
    * 정의역: $\frac{1+x}{x} \ge 0$ 이고 $x \ne 0$ 이어야 함. 
    * 결과: $\{x \mid x \le -1 \text{ 또는 } x > 0\}$
* **$g(f(x))$:** $g(\sqrt{x+1}) = \frac{1}{\sqrt{x+1}}$
    * 정의역: 분모가 0보다 커야 하므로 $x+1 > 0$
    * 결과: $\{x \mid x > -1\}$

#### 2. (연습문제 1.1-8) 합성함수 형태로 표현
$F(x) = (f \circ g \circ h)(x)$ 형태로 표현 가능합니다.
* $h(x) = x^3 - 1$
* $g(x) = |x| + 3$
* $f(x) = \sqrt{x}$
* 즉, $F(x) = f(g(h(x)))$

#### 3. (연습문제 1.2-8) 자연로그 역함수

2.  $\ln y = x - 1$
3.  $x = \ln y + 1$
4.  변수 교체: $f^{-1}(x) = \ln x + 1$

#### 4. (연습문제 1.3-1) 쌍곡함수 계산
* $\sinh(\ln 2) = \frac{e^{\ln 2} - e^{-\ln 2}}{2} = \frac{2 - 1/2}{2} = \frac{3}{4}$
* $\cosh(\ln 3) = \frac{e^{\ln 3} + e^{-\ln 3}}{2} = \frac{3 + 1/3}{2} = \frac{5}{3}$
* 값: $\frac{3}{4} + \frac{5}{3} = \frac{9+20}{12} = \frac{29}{12}$

#### 5. (연습문제 1.4-6) 역삼각함수
$\cos^{-1}(4/5) = \alpha$, $\cos^{-1}(3/5) = \beta$라 하면:
* $\cos \alpha = 4/5 \rightarrow \sin \alpha = 3/5$
* $\cos \beta = 3/5 \rightarrow \sin \beta = 4/5$
* $\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta = (3/5)(3/5) + (4/5)(4/5) = \frac{9+16}{25} = 1$

#### 6. (예제 1.4-15) 역쌍곡사인함수 증명
$y = \sinh^{-1}x \iff x = \sinh y = \frac{e^y - e^{-y}}{2}$
1.  $2x = e^y - \frac{1}{e^y}$
2.  $(e^y)^2 - 2x(e^y) - 1 = 0$ (근의 공식 사용)
3.  $e^y = \frac{2x \pm \sqrt{4x^2 + 4}}{2} = x \pm \sqrt{x^2 + 1}$
4.  $e^y > 0$ 이므로 $e^y = x + \sqrt{x^2 + 1}$
5.  $y = \ln(x + \sqrt{x^2 + 1})$

#### 7. (연습문제 3.1-1) 연속성과 미분가능성
* **연속성:** $\lim_{x \to 0} x^2 \sin(1/x) = 0$ (샌드위치 정리). $f(0)=0$이므로 **연속**.
* **미분가능성:** $f'(0) = \lim_{h \to 0} \frac{h^2 \sin(1/h) - 0}{h} = \lim_{h \to 0} h \sin(1/h) = 0$. **미분 가능**.

#### 8. (연습문제 3.1-11) 절댓값 도함수
$|x| = (x^2)^{1/2}$ 이므로 체인룰 적용:
$$\frac{d}{dx}(x^2)^{1/2} = \frac{1}{2}(x^2)^{-1/2} \cdot 2x = \frac{x}{\sqrt{x^2}} = \frac{x}{|x|}$$
분모 분자에 $|x|$를 곱하면 $\frac{x|x|}{|x|^2} = \frac{x|x|}{x^2} = \frac{|x|}{x}$

#### 12. (연습문제 3.4-1) 평균값 정리
$f'(c) = \frac{f(2)-f(-1)}{2-(-1)} = \frac{3-0}{3} = 1$
$3c^2 - 2c - 1 = 1 \rightarrow 3c^2 - 2c - 2 = 0$. 근의 공식으로 $c$ 계산 (구간 내 존재 확인).

#### 13. (연습문제 3.4-6) 평균값 정리 극한
$f(t) = 3^t$라 하면 $\frac{3^x - 3^{\sin x}}{x - \sin x} = f'(c)$ ($c$는 $\sin x$와 $x$ 사이)
$x \to 0^+$일 때 $c \to 0$. $f'(t) = 3^t \ln 3$ 이므로 결과는 $3^0 \ln 3 = \ln 3$.

#### 15. (연습문제 4.4-8) l'Hopital 법칙
$L = \lim_{x \to 0}(1-\tan x)^{1/x}$
$\ln L = \lim \frac{\ln(1-\tan x)}{x} = \lim \frac{\frac{-\sec^2 x}{1-\tan x}}{1} = -1$
$L = e^{-1} = 1/e$

#### 23. (연습문제 5.3-5) 치환적분
$u = 2x+2, du = 2dx$. 구간 $[0, 2] \rightarrow [2, 6]$
$\int_0^2 f(2x+2) dx = \frac{1}{2} \int_2^6 f(u) du = \frac{1}{2}(12) = 6$
